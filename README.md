# Sobre

Acompanhe a contagem de votos da eleição presidencial no Brasil

## Instalação

```bash
$ git clone https://github.com/RubenFontes/brazil-apuracao-2022.git
```

```bash
$ cd brazil-apuracao-2022
```

```bash
$ npm install
```

## Inicializar

```
$ node app.js
```
