# Sobre

Acompanhe a contagem de votos da eleição presidencial no Brasil

## Instalação

```bash
$ git clone https://github.com/RubenFontes/api-eleicoes-br.git
```

```bash
$ cd api-eleicoes-br
```

```bash
$ npm install
```

## Inicializar

```
$ node app.js
```
